First paragraph. Use noindent class if you want to.

Second paragraph. Never use noindent class on a paragraph that’s not first in a block of paragraphs.

“This is some dialogue in English.” Use curly quotation marks! Note that they are different if they are at the start or end. For words like [that’s], [rock ’n’ roll] and [’bout] make extra sure that it's starting with an apostrophe (’).

“Use em-dashes to end something abruptl—”

Instead of using two, three or four periods (... or sometimes even . . .), replace them with the ellipsis (…) character. Sigil has a Insert Special Character button that’s very useful. But don’t go and blindly do a Replace All without checking what you replace!
